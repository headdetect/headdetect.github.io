# headdetect.github.io

A personal site that does exactly one thing: tells you who I am and what I do.

Inspired by [Motherfucking Website](https://motherfuckingwebsite.com/): if my personal site needs a
bundler, I've already lost.

## Run it

```sh
python3 -m http.server 8000
```

Open [http://localhost:8000](http://localhost:8000).

That's it
